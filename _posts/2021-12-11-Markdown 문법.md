## Markdown
마크다운은 마크업 언어의 일종으로, 존 그루버(John Gruber)와 아론 스워츠(Aaron Swartz)가 만들었다.
많은 태그를 작성해야하는 HTML문서 등과 달리 읽기도 쓰기도 쉬운 양식을 지향한다.
확장자는 `.md` 또는 `.markdown`을 쓸 수 있다. 보통 전자를 많이 사용한다.

## 문법
### 문단제목 (Header)
```
# 문단 제목
## 단계가 커질수록 점점 작아지는데
### 총 6단계까지 가능하다.
```
### 기울임체 (Italic)
```
*...*
_..._
```
_적용결과_

### 강조체 (Bold)
```
**...**
__...__
```
**적용결과**

### 순서 없는 리스트 (Unordered list)
```
- ...
* ...
```
- 적용결과
- 한 칸 띄우는것 잊지말기 **(중요!)**

### 순서 리스트 (Ordered list)
```
1. 이렇게 써도
2. 이렇게
5. 보인다
```
1. 이렇게 써도
2. 이렇게
5. 보인다

### Code, Code Block
```

```
